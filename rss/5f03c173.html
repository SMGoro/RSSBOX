    Podman 5.0 正式发布

    Podman 5.0 已正式发布。新版本针对 Windows 和 Mac 平台完全重写了代码，并对这两个平台上的虚拟机管理程序支持进行了重大改进。

    Podman 5.0 的一大亮点是对 podman machine 命令的彻底重写。该功能允许用户在 Windows 和 Mac 系统上启动 Linux 虚拟机，使他们能够无缝运行 Linux 容器。

    Linux 6.9 弃用 ext2 文件系统驱动程序

    在即将发布的 6.9 Linux 内核中，ext2 文件系统驱动程序将被标记为已弃用。EXT2 第二代扩展文件系统是 Linux 内核所用的文件系统，最开始由 Rémy Card 设计用以代替 ext，于 1993 年 1 月加入 Linux 核心支持之中，至今已有三十多年的历史。
